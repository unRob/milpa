milpa itself version

Displays the version of milpa

Usage

#

milpa itself version [options]

Description

#

Outputs the version of milpa to stdout, printing a new line if the output is a TTY. Also available as milpa --version

Global Options

#
  • --help (bool): Display help for any command.
  • --no-color (bool): Print to stderr without any formatting codes.
  • --silent (bool): Silence non-error logging.
  • --verbose (bool): Log verbose output to stderr.
