milpa itself

itself subcommands

Usage

#

milpa itself [flags] subcommand

Subcommands

#
  • create - Create a milpa command
  • docs - docs subcommands
  • doctor - Validates all commands found on the MILPA_PATH
  • repo - repo subcommands
  • shell - shell subcommands
  • version - Displays the version of milpa

Description

#

Runs subcommands within itself

Global Options

#
  • --help (bool): Display help for any command.
  • --no-color (bool): Print to stderr without any formatting codes.
  • --silent (bool): Silence non-error logging.
  • --verbose (bool): Log verbose output to stderr.
