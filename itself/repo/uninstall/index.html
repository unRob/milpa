milpa itself repo uninstall

Removes an installed milpa repo

Usage #

milpa itself repo uninstall [options] PATH

Arguments #

  • PATH - The repo path to uninstall

Description #

Uninstalls a milpa repo by PATH. See milpa itself repo list --help for a list of available repos

Global Options #

  • --help (bool): Display help for any command.
  • --no-color (bool): Print to stderr without any formatting codes.
  • --silent (bool): Silence non-error logging.
  • --verbose (bool): Log verbose output to stderr.
