repo subcommands

Usage #

milpa itself repo subcommand

Subcommands #

  • install - Installs a milpa repo
  • list - Lists installed repos
  • uninstall - Removes an installed milpa repo

Description #

Runs subcommands within repo

Global Options #

  • --help (bool): Display help for any command.
  • --no-color (bool): Print to stderr without any formatting codes.
  • --silent (bool): Silence non-error logging.
  • --verbose (bool): Log verbose output to stderr.
